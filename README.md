## Getting Started

### Prerequisites

- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   https://github.com/Jkitche/rps-hunt.git
   ```
2. Install NPM packages
   ```sh
   yarn install
   ```
   or
   ```
   npm install
   ```
3. Run the app
   ```sh
   yarn dev
   ```
   or
   ```sh
   npm run dev
   ```
